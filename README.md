# Customer Churn Prediction Project

## Overview
This workspace contains a customer churn prediction analysis project for the Telco Customer Churn dataset. The main work is located under `KHUSHI_SKS_DATA ANALYSIS INTERNSHIP/KHUSHI_SKS_DATA ANALYSIS INTERNSHIP_SUBMISSION/k_final`.

## Contents
- `KHUSHI_SKS_DATA ANALYSIS INTERNSHIP/KHUSHI_SKS_DATA ANALYSIS INTERNSHIP_SUBMISSION/k_final/Telco_Customer_Churn_Dataset_.csv`
  - Original customer churn dataset.
- `cleaned_churn_data.csv`
  - Data after cleaning and preprocessing.
- `ml_ready_churn_data.csv`
  - Dataset prepared for machine learning modeling.
- `segmented_churn_data.csv`
  - Customer segments and clusters.
- `customers_with_ltv_scores.csv`
  - Customer lifetime value scoring results.
- `high_value_at_risk_customers.csv`
  - Identified high-value customers who are at risk of churn.

## Notebooks
- `Task1_Data_Cleaning.ipynb`
  - Data cleaning and preprocessing.
- `Task2_EDA.ipynb`
  - Exploratory data analysis and visualizations.
- `Task3_Segmentation.ipynb`
  - Customer segmentation analysis.
- `Task4_Model.ipynb`
  - Machine learning model training and evaluation.
- `Task5_Retention.ipynb`
  - Retention analysis and strategies.
- `Task6_Visualisations.ipynb`
  - Final visualizations and summary charts.

## How to Use
1. Open the workspace in VS Code.
2. Open the notebooks in `KHUSHI_SKS_DATA ANALYSIS INTERNSHIP/KHUSHI_SKS_DATA ANALYSIS INTERNSHIP_SUBMISSION/k_final`.
3. Run the notebooks in order from Task 1 through Task 6.

## Notes
- This project appears to be structured for an internship submission.
- The analysis is based on the Telco Customer Churn dataset and focuses on churn prediction, segmentation, and retention.
